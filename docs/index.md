# Api


## Formato

_response_

```json
{
    "data": {}, // array o objecto
    "message": "Mensaje de error o comentario del requerimiento",
    "success": true // estado del requerimiento
}
```

## [Login](#login)

Nombre: login 

> POST /api/login

_request_

```json
{
	"email": "marc34@example.com",
	"password": "123456"
}
```

_response_

```json
{
  "data": {
      "access_token": "",
      "token_type": "bearer",
      "expires_in": 3600
  },
  "success": true
}
```

## [Logout](#logout)

Nombre: logout 

> POST /api/logout

_response_

```json
{
  "message": "Successfully logged out",
  "success": true
}
```

Nombre: register

```json
{
	"email": "cupuyaa@clickmail.info",
	"password": "12dsds",
	"names": "joel",
	"surnames": "joel"
}
```

```json
{
  "success": true,
  "data": {
      "email": "cupuyaa@clickmail.info",
      "names": "joel",
      "user_type_id": 1,
      "surnames": "joel",
      "updated_at": "2019-06-07 00:26:46",
      "created_at": "2019-06-07 00:26:46",
      "id": 61
  }
}
```
